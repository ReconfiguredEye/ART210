Unit 2 | Workshop

The Body Extended

Getting Started

  • Find an image containing the human form with a simple background
  • Separate the human figure from the background using Photoshop or a similar bitmap image editor. There should be no visual artifacts left from this separation
  • Load both images into the web editor for p5js and draw them as if they are a single image

Goals

Create three compositions transfiguring the body into gemoetrical shapes, or equivalently, a living sculpture.

  1. Minimize the subject matter
  2. Extend the subject matter
  3. Create a combination of the first two items in this list

    4. Requirements
    • use at least three different shapes (at least one use of "vertex()" and / or "curveVertex())"
    • use of a "for loop"
    • use buttons to allow the user to toggle on / off (can be either html or JavaSrcipt buttons) - character, background, reference
    • well commented and organized code
